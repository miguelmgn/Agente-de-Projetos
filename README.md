# Agente-de-Projetos
Arquivos MD para uso em LLMs
Nos últimos tempos reformulei a forma como usava IA. Com acesso em algumas comunidades  me deparei 
com a tese de que montar agentes com memória anexa em arquivo Markdown eram mais mais econômicos e 
seus resultados gerados surtiam maiores proveitos. Assim o fiz.

Os Arquivos em MD me permitem estrutura um escopo mais elaborado maior e deixa-lo anexado permite que o 
agente o qual estou trabalhando opere de forma mais específica. 

Outra vantagem foi a possibilidade em manutenções nas diretrizes sem ter que reenviar prompts via chat - Tenho atualizações a fazer? 
abro o arquivo, estruturo as novas linhas e subo o novo modelo no campo de memória.
